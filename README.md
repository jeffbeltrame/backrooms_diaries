# Projeto O Diário das Backrooms (Arquivos Perdidos)

> **⚠️ Aviso:** Por favor, leia todo este README para entender o contexto e o propósito deste repositório.

## Sobre o Projeto

Este projeto foi criado originalmente por **Jefferson Luiz Beltrame** na data de **16/12/2022**. 

A ideia inicial nasceu com dois propósitos principais:
1. Colocar em prática e testar conhecimentos sobre as linguagens de marcação **HTML** e **CSS**.
2. Criar um site que funciona como uma grande "coletânea" de arquivos perdidos (diários, panfletos, páginas, recados, áudios e vídeos) supostamente encontrados nas Backrooms.

## Evolução e Propósito Atual (Abril de 2026)

## O que são as Backrooms?

Para aqueles que não estão familiarizados com o fenômeno:

> *As "Backrooms" começaram como uma lenda urbana da internet. Elas descrevem um labirinto interminável e não-linear de salas de escritório vazias. O ambiente é caracterizado pelo cheiro de carpete úmido, paredes com papel de parede amarelo monocromático e o zumbido irritante e constante de velhas lâmpadas fluorescentes. A lenda diz que, se você cometer um erro na realidade e "atravessar" o chão, acabará preso nessa dimensão, que é dividida em milhares de níveis repletos de mistérios.*

**Link importante para entender a história:**
* [The Backrooms Wikidot](http://backrooms-wiki.wikidot.com/)
